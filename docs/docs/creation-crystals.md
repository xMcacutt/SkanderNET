# The Creation Crystal Figure

> [!NOTE]
> Many properties and methods from <xref:SkanderNET.Figures.CreationCrystalFigure> are omitted in this document.
> They are self-explanatory. More info in the API section.


CYOS stands for Create Your Own Skylander.

Every customization has its own enum except the scale values and the catchphrases which are simply values.
The scale values have a maximum of 8.

The battle class of imaginators can be changed but might need to be saved twice.

The weapon of imaginators does not need to match the battle class. However, the weapon will still act as though it is the type of weapon belonging to the battle class and will not display.

The upgrade status of abilities is seemingly absent from the figure suggesting it is saved on the game save file.
